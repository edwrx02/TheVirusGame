The Virus 
Programming by Edwin Quintanilla
Art by Adam Patrick
How to play:
	Move the player with the arrow keys
	Up = UP key
	Down = Down key
	Left = Left key
	Right = Right key
	To shoot use the space bar
	Shoot = SpaceBar
Kill as many viruses without dying.
you have 6 health points.
There are 3 stages. 
Stage 1 is easy, enemies are slow.
Stage 2 is medium, enemies are fast.
Stage 3 is hard, the boss.
